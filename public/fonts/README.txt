Inter Font Setup
================

To complete the self-hosted font setup:

1. Download Inter-Variable.woff2 from:
   https://github.com/rsms/inter/releases
   
2. Place the file in this directory:
   public/fonts/Inter-Variable.woff2

The font is licensed under SIL Open Font License 1.1
https://github.com/rsms/inter/blob/master/LICENSE.txt

Benefits of self-hosting:
- No external dependencies (privacy)
- Better performance (no DNS lookup)
- Works offline
- CSP compliant
- GDPR compliant (no Google tracking)

File size: ~320KB (but cached forever after first load)
